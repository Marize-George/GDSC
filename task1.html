Viola
WorldWideWeb
Hypermedia Browser

ViolaWWW is a World Wide Web browser. ViolaWWW is built using the Viola
hypermedia language / toolkit , and now also comes with a Motif front end

Viola's support of HTML 3.0 (aka HTML+) so far includes:

  • Paragraph as container
  • Nesting lists
  • Input forms
  • Tables
